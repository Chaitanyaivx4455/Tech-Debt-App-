# Tech-Debt-App

Technical Debt Tracker is a full-stack web application built with Next.js, MongoDB, and Tailwind CSS. It helps development teams track, prioritize, and manage technical debt through debt logging, status tracking, and workflow management, improving code quality and project maintainability.

## Features

* Add Technical Debt
* Edit Technical Debt
* Delete Technical Debt
* Priority Management (Low, Medium, High)
* Status Tracking (Open, In Progress, Resolved)
* MongoDB Database Integration
* Responsive UI with Tailwind CSS

## Tech Stack

* Next.js
* React
* MongoDB
* Mongoose
* Tailwind CSS
* TypeScript

## Installation

```bash
npm install
npm run dev
```

Open http://localhost:3000 in your browser.
